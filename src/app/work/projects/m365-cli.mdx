## What this is

A CLI tool for Microsoft 365 governance and security operations. Built for speed and automation — no clicking through admin centers.

## What's in it

- **Entra ID commands** — User lifecycle, group management, role assignments
- **Intune commands** — Device compliance, configuration profiles, app deployment
- **Defender commands** — Alert triage, incident response, threat hunting
- **SharePoint commands** — Site permissions, content audit, drive management
- **Bulk operations** — Process thousands of objects at once

## Why it matters

The Microsoft 365 admin centers are slow and don't scale for automation. This CLI gives you scripts and pipelines that can handle enterprise-scale operations.

## Stack

- Rust (core TUI)
- TypeScript / Bun (workers)
- Microsoft Graph API
- OIDC authentication
