# FrontendDesign - Modern UI/UX Engineering

**Invoke when:** designing React components, optimizing frontend performance, implementing state management, handling forms, or ensuring accessibility.

## Overview

The FrontendDesign skill encapsulates the "Gold Standard" for UI engineering within this project. It enforces:
- **Component Composition** over inheritance.
- **Hook-based Logic** for cleaner state management.
- **Performance-First** rendering (memoization, virtualization).
- **Accessibility** as a first-class citizen.

## Core Components

### 1. Context-FrontendPatterns.md
The definitive guide to approved patterns, including:
- **Composition**: Compound components, render props.
- **State**: Context + Reducer, custom hooks (useToggle, useQuery).
- **Performance**: Virtualization, lazy loading, memoization strategies.
- **UX**: Framer Motion animations, accessible modal management.
